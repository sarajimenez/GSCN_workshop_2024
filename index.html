GSCN workshop 2024#

This repository is a collection of tutorials presented during the GSCN workshop 2024 ‘Computational single cell genomics for stem cell biologists’ - November 7-8, 2024.

The corresponding Jupyter Book is rendered here.

Material#

The data for each session can be found here.
Material for the workshop has been adapted from the single-cell best practices book, a live project from the Theis lab.
Recordings of previous single-cell RNA seq data analysis workshops can be found here.
